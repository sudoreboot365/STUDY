**VLANs + ROAS**
- Switches
	- create vlans: 
		- `vlan 10`, `vlan 20`, etc
	- for each interface to PCs:
		- `switchport mode access`
		- `switchport access vlan 10`
	- interface to switches:
		- `switchport mode trunk`
		- `switchport trunk allowed vlan 10,20`
		- `switchport trunk native vlan 1001`
		- `vlan 30`

	- interface to ROAS:
		- `switchport mode trunk`
		- `switchport trunk allowed vlan 10,20`
	- check 
		- `do show interfaces status`
		- `do show vlan brief`
		- `do show interfaces trunk`
- Router
	- parent interface
		- `no shutdown`
	- for each vlan, create subinterfaces
		- `interface g0/0.10`
		- `encapsulation dot1q 10`
		- `ip address 192.168.10.lastUseableHost 255.255.255.0`
	- check
		- `do show ip int brief`
- PCs
	- ip addresses in subnets
		- ex. vlan 10 = 192.168.10.host/24
	- default gateway: ip address of corresp subinterface
		- ex. vlan 10 = 192.168.10.lastUseableHost
	- check:
		- ping other pics in other vlans

**Static Routes**
- Router
	- Set IPs on interfaces
		- `ip address 192.168.network.host 255.255.255.0`
		- `no shutdown`
	- Check
		- `do show ip int brief`
	- Set static routes
		- `ip route 192.168.destNetwork.0 255.255.255.0 192.168.p2pNet.nextHop`
			- nextHop = IP of ingress interface on next router
		- `ip route 192.168.destNetwork.0 255.255.255.0 g0/0`
	- Check
		- `do show ip route`
